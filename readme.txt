A simple, light-weight WordPress plugin that allows you to easily add shareable quotes via shortcodes.

== Description ==

Use the following shortcode to include shareable quotes:

    [quoteability]I love the Quoteability WordPress plugin! -Ben Marshall[/quoteability]

Available options:

* via - Adds @VIA to quote.
* button - Share link text (defaults to 'Tweet This Quote')
* class - Class for blockquote
* override - Quote text (defaults to the text within the shortcode)
* url - URL to share (defaults to current url)
* target - Defaults to '_blank'

**Languages:** English

== Installation ==

= Plugin Installation =

1. Upload the `quoteability` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
